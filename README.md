# FlashBangROM
Firmware &amp; GUI for flashing ROMs with inexpensive hardware. The focus is programming SST39 flash chips with a BluePill (STM32) board. The BluePill must be modified to get 30 GPIOs. Future plans are to include other flashchips (Winbond?) and maybe later to bring this to BlackPill Boards that shouldn't  need to be modified. This is a hobby project!
